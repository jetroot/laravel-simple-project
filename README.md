## About Project

A project for creating posts whith login and sign up.

## Install && Running the database

1. Install dependicies
```bash
npm install
```
2. Compile dependicies
```bash
npm run dev
```
3. Run database
```bash
cd docker && docker compose up
```

## Running the server
```bash
php artisan serve
```

## Try on browser
```bash
http://127.0.0.1:8000/
```
